Funciones del proceso administrativo
  1. Planear
  2. Organizar
  3. Dirigir
    1. Se debe de buscar el ganar - ganar siempre
    2. Se requiere mejorar las habilidades blandas
  4. Controlar
    1. Lo que no se mide no se controla
    2. Se debe de dar seguimiento

Todos estos pasos pueden y deben de ser seguidos en nuestra pequeña empresa, conformada por los diferentes grupos de trabajo.
Es importante saber dar y recibir retroalimentación, de esa forma todos podremos mejorar, además de que en los grupos se deberán de llenar diferentes calificaciones entre los integrantes.
En mi trabajo utilizamos el sistema de tiquetería para poder controlar las solicitudes recibidas, la cantidad de trabajo que atiende cada desarrollador, la cantidad de trabajo por departamente, y también se recibe una calificación del cliente al resolver el tiquete.
Además de eso, se utiliza Azure DevOps para llevar el control de las historias de usuario que tiene asignadas cada desarrollador, así como sus tareas, estimaciones, y estado actual. Esto es importante para saber cómo le fue al equipo durante el sprint anterior y poder mejorar en el siguiente.
